アップグレード

古いJekyllからのアップグレードですか？　新しいメジャーバージョンへのアップグレード（v2.xからv3.xなど）は頭痛の種のになるかもしれません。以下のガイドに従って、アップグレードを進めてください。

  • 0.xから1.xや2.xへ
  • 2.xから3.xへ
  • 3.xから4.xへ

マイナーアップデート

最新版を維持してください

最新のバグの修正の恩恵を受けるために、できるだけ頻繁にJekyllをアップデートすることをお勧めします。

セットアップ時の推奨に従ってBundlerをインストールした場合は、bundle update jekyllやもっとシンプルにbundle updateを実行してください。全てのgemが最新版にアップデートされます。

Bundlerをインストールしていなければ、gem update jekyllを実行してください。

github-pages gemを使用している場合も同様です。
